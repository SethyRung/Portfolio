# Asset Management System

## Overview

The Asset Management System is a web application built with **Nuxt.js** for the frontend and **Spring Boot** for the backend. It provides functionalities to manage assets, track their lifecycle, and generate reports.

## Tech Stack

- **Frontend:** Nuxt.js, Nuxt UI (UI library)
- **Backend:** Spring Boot, Spring Security (JWT Authentication), Spring Data JPA
- **Database:** PostgreSQL

## Features

### Authentication & Authorization

- User login using email & JWT authentication
- Role-based access control

### Asset Management

- Add, edit, delete assets
- Assign assets to employees
- Track asset status (active, inactive, under maintenance)
- Maintain asset categories

### User Management

- Register users
- Assign roles to users
- Manage user permissions

### Reports & Auditing

- Generate asset usage reports
- Track asset history
