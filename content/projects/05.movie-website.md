# Movie Website React

A responsive movie website built with React and TypeScript, allowing users to explore a vast collection of movies and TV shows with an intuitive interface and seamless navigation.

## Features

- **Search Functionality**: Easily search for movies and TV shows.
- **Multiple Pages**
- **Dark Mode Toggle**
- **Responsive Design**
- **Animations**

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Getting Started

Follow these instructions to set up and run the project locally.

## Installation

1. **Clone the repository**:

```bash
git clone https://github.com/SethyRung/Movie-Website-React.git
Navigate to the project directory:
```

2. **Development**:

```bash
npm install
npm run dev
```
